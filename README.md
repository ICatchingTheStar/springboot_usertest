# 关于该项目

这是基于springboot框架的简易项目：用户管理系统，主要由**springboot2.3.7**、**mybatis-plus3.4.2**、**mysql5.1.49**以及前段模板引擎**thymeleaf**完成。 

# 大致结构

```mermaid
graph LR
A[登录功能] -- 欢迎页跳转 --> B((用户表))
B --> D{分页展示}
B --> E{增删改}
B --> F{批量删除}
B --> G{复杂条件查询}
D --> H{数据库}
E --> H{数据库}
F --> H{数据库}
G --> H{数据库}
```
